# Yamas – Luxury Greek Restaurant Website

Premium dark Mediterranean restaurant website for Yamas, Clausthal-Zellerfeld.

## Run locally

```bash
npm install
npm run dev
```

## Media assets (add to `public/`)

| Path | Purpose |
|------|---------|
| `videos/yamas-hero.mp4` | Full-screen cinematic hero video |
| `images/yamas-hero-poster.jpg` | Hero fallback poster |
| `images/gyros-premium.jpg` | Signature dish |
| `images/souvlaki-premium.jpg` | Signature dish |
| `images/salad-premium.jpg` | Signature dish |
| `images/pita-premium.jpg` | Signature dish |
| `images/interior.jpg` | Intro section |
| `images/breakfast.jpg` | Breakfast section |

Until files are added, elegant gradient placeholders are shown automatically.

## Design system

- **Midnight** `#06111F` · **Charcoal** `#0B0B0B` · **Ivory** `#F8F1E7`
- **Champagne** `#C9A45C` · **Olive** `#6F7D45` · **Terracotta** `#B85C38`
- Typography: Playfair Display + Inter (via `next/font/google`)
